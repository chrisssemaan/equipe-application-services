# Team Management Application

Is an application to Manage Nice Teams

### Prerequisites

- Java 8 and above
- Maven

### Installing

######Checkout the project from git

```
git clone path 
```

######To run the application
```
mvn spring-boot:run
```

######To run tests

```
mvn clean verify
```
